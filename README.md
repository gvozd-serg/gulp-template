#### Installation
- > npm i

#### Development
- > gulp

#### Build
- > gulp build

#### Deploy to FTP

- > Enter to gulpfile.js
    host:      '',
    user:      '',
    password:  '',
- > gulp build
- > gulp deploy

#### Start server on the 8080 port
- > http://localhost:8080/

#### Svg to icon-font generator

1. add to folder 'icons' svg files
2. - > gulp icon
3. Add to project sass/_icons.scss

#### Npm latest
- > npm install npm@latest -g
